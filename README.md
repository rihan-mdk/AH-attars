## 🏛️ About the Project

**AH attars** is a high-end e-commerce platform designed for fragrance lovers. It bridges the gap between traditional perfumery and modern minimalist fashion. Built with a focus on **visual excellence** and **smooth performance**, the application offers a seamless experience from scent discovery to checkout.

## ✨ Key Features

### 💎 For Customers
*   **Dynamic Discovery**: Explore a curated catalog of fragrances and premium apparel.
*   **Intelligent Pricing**: Real-time price calculation based on chosen variants (e.g., 6ml, 12ml, 1 Thola) and quantity.
*   **Global Currency**: Native support for **Indian Rupees (₹)** with automatic local detection.
*   **Seamless Checkout**: Integrated address management and secure checkout flow.
*   **Aesthetic UI**: A premium interface featuring "Outfit" typography and glassmorphism design elements.

### 🔐 For Administrators
*   **Smart Dashboard**: Manage your entire catalog with a dedicated admin suite.
*   **Variant Management**: Add, update, or remove custom sizes and pricing for any product on the fly.
*   **Secure Access**: Role-based security ensuring only authorized administrators can modify the catalog.
*   **Order Tracking**: Monitor customer orders and business performance.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Motion (Framer Motion)
- **Backend/Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **Icons**: Lucide React
- **Hosting**: Vercel / AI Studio

## 🚀 Getting Started

### Prerequisites
*   [Node.js](https://nodejs.org/) (v18 or higher)
*   [Firebase Account](https://console.firebase.google.com/)

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/your-username/aura-fragrances.git
    cd aura-fragrances
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    *   Rename `.env.example` to `.env.local`.
    *   Populate your Firebase configuration and API keys.

4.  **Run Development Server**
    ```bash
    npm run dev
    ```

## 📜 Security & Privacy
This project uses **Firestore Security Rules** to ensure that customer data is isolated and administrative actions are strictly verified. All sensitive API keys are managed through environment variables to prevent exposure.
